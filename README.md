# Russian Crowdfunding Success Factors

Проект посвящен анализу факторов успеха краудфандинговых проектов в России на основе данных платформы Planeta.ru.

Цель проекта — выявить, какие характеристики проектов, авторов, текстового описания и кампании связаны с успешностью сбора средств. В работе используются методы обработки данных, EDA, эконометрики и ML.

## Описание проекта

Краудфандинг в России развивается как инструмент альтернативного финансирования, однако доля успешных проектов остается ниже, чем на крупнейших зарубежных платформах. Это делает важным анализ факторов, которые помогают проектам достигать заявленной цели сбора.

В проекте рассматриваются следующие группы факторов:

* характеристики проекта: категория, цель сбора, длительность кампании, число бэкеров, сумма сборов;
* характеристики автора: опыт, число проектов, успешность прошлых кампаний;
* характеристики текстового описания: длина, тональность, читаемость, наличие смысловых и доверительных сигналов;
* активность вокруг проекта: комментарии, новости, обновления, реакции аудитории;
* финансовые показатели: средний взнос, доля собранной суммы, достижение целевой суммы.

## Исследовательские вопросы

* Какие характеристики проектов связаны с успешностью сбора средств?
* Какие текстовые признаки описания проекта влияют на вероятность успеха?
* Какие категории проектов чаще достигают целевой суммы?
* Как распределяются проекты по суммам сбора, числу бэкеров и уровню достижения цели?
* Можно ли предсказывать успешность проекта на основе табличных и текстовых признаков?

## Файлы проекта

### `Planeta.ipynb`

Ноутбук содержит обработку сырых данных, подготовку первичных признаков и блок визуализаций. Используется как один из первых этапов пайплайна после парсинга данных с платформы.

### `outlier_cleaning.ipynb`

Ноутбук посвящен очистке данных от выбросов. На этом этапе анализируются аномальные значения по финансовым и поведенческим признакам, после чего формируется более стабильная выборка для дальнейшего анализа.

### `eda_and_data_cleaning.ipynb`

Ноутбук содержит разведочный анализ данных и дополнительную очистку. В нем исследуются распределения ключевых переменных, структура категорий, пропуски, а также связи между признаками и успешностью проектов.

### `texts_experiments.ipynb`

Ноутбук содержит эконометрический анализ и ML-эксперименты. В нем проверяются гипотезы о влиянии проектных и текстовых характеристик на успешность краудфандинговых кампаний.

### `raw files/planeta_projects_with_features.xlsx`

Исходный датасет с проектами Planeta.ru и рассчитанными признаками. Используется как основа для очистки, EDA, визуализаций и моделирования.

## Данные

Основной датасет содержит информацию о краудфандинговых проектах Planeta.ru.

Примеры признаков:

* название проекта;
* категория проекта;
* целевая сумма сбора;
* фактически собранная сумма;
* доля выполнения цели;
* количество бэкеров;
* длительность кампании;
* текстовое описание проекта;
* признаки текстового контента;
* признаки активности автора и аудитории;
* бинарный индикатор успешности проекта.

## Методология

Работа состоит из нескольких этапов:

1. Сбор данных с платформы Planeta.ru.
2. Обработка и объединение сырых данных.
3. Очистка пропусков и выбросов.
4. Разведочный анализ данных.
5. Визуализация распределений и связей между признаками.
6. Генерация и анализ текстовых признаков.
7. Эконометрическое моделирование.
8. ML-эксперименты для предсказания успешности проектов.
9. Интерпретация результатов.
